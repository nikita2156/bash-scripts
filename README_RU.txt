================================================================================
                    SSH Tunnel Creator для Windows
================================================================================

ОПИСАНИЕ:
---------
Простое десктопное приложение для быстрого создания SSH туннелей.
Совместимо с Windows 7 и выше.

ВОЗМОЖНОСТИ:
------------
- Local Forward (L) - перенаправление локального порта на удалённый сервер
- Remote Forward (R) - перенаправление удалённого порта на локальный сервис
- Dynamic Forward (D) - SOCKS5 прокси через SSH туннель
- Поддержка аутентификации по паролю и SSH ключам
- Встроенный лог событий
- Простой и понятный интерфейс

ТРЕБОВАНИЯ:
-----------
- Windows 7 или выше
- Python 3.6 или выше (для запуска из исходного кода)
- Библиотека paramiko (устанавливается автоматически при сборке)

УСТАНОВКА И ЗАПУСК:
-------------------

ВАРИАНТ 1: Запуск из исходного кода
1. Установите Python с https://www.python.org/downloads/
2. Откройте командную строку в папке с приложением
3. Выполните: pip install -r requirements.txt
4. Запустите: python ssh_tunnel_app.py

ВАРИАНТ 2: Сборка в исполняемый файл (.exe)
1. Установите Python с https://www.python.org/downloads/
2. Откройте командную строку в папке с приложением
3. Запустите: build.bat
4. Готовый файл будет в папке: dist\SSH_Tunnel_Creator.exe

ИСПОЛЬЗОВАНИЕ:
--------------
1. Введите адрес SSH сервера, порт (по умолчанию 22)
2. Введите имя пользователя и пароль (или путь к SSH ключу)
3. Настройте нужный тип туннеля:
   - Local Forward: укажите локальный порт и назначение (хост:порт)
   - Remote Forward: укажите удалённый порт и назначение
   - Dynamic Forward: укажите порт для SOCKS прокси
4. Нажмите "Подключиться"
5. Для отключения нажмите "Отключиться"

ПРИМЕРЫ ИСПОЛЬЗОВАНИЯ:
----------------------

Local Forward:
  Локальный порт: 8080
  Назначение: internal-server:80
  Результат: http://localhost:8080 -> internal-server:80 через SSH

Dynamic Forward (SOCKS):
  Порт: 9050
  Результат: SOCKS5 прокси на localhost:9050
  Настройте браузер на использование прокси SOCKS5 localhost:9050

TROUBLESHOOTING:
----------------
- Ошибка "paramiko не установлена": выполните pip install paramiko
- Ошибка подключения: проверьте доступность SSH сервера и корректность учётных данных
- Туннель не работает: убедитесь, что локальный порт не занят другим приложением

ЛИЦЕНЗИЯ:
---------
Приложение распространяется свободно. Используйте на свой страх и риск.
